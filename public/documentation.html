Business Logic URL HTTP Method Request body data format Respones body data format
Get list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing the ID, title, description, genre, director, and image URL. Example
{
 id: 1,
 title: "Oldboy",
 description: "Thriller about revenge.",
 genre: [
     'Thriller',
     'Drama'
     ],
 director: "Park Chan-Wook",
 image URL: https://www.urlofmovie.com
}
Get data about a genre by name/title /movies/[Genre] GET None A JSON object holding data about the genre, structured like:
{
 Genre: "Thriller",
 Movies: [
     'Oldboy',
     'Parasite'
     ],
}
Get data about a director (bio, birth year, death year) by name /movies/[Director] GET None A JSON object holding data about the director, structured like:
{
 Name: "Park Chan-Wook",
 Birth year: 1963
 Death year: "Still alive"
}

Update the username of a user /users/[id]/[Username] PUT None A text message indicating whether the username was updated
Allow a user to add a movie to their list of favorites /users/[Name]/[id]/movies/[Title] POST None A text message indicating whether the movie has been added
Allow a user to remove a movie from their list of favorites /users/[Name]/[id]/movies/[Title] DELETE None A text message indicating whether the movie has been removed
Allow a user to deregister /users/[Name] DELETE NONE A text message indicating whether the user info has been deleted
